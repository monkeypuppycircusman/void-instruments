The Instruments
Under the Hood
Physical Modeling
Reed-bore feedback, bow stick-slip, Karplus-Strong strings, edge-tone vortex shedding, stopped cylinder resonance.
Web Audio API
Every sound runs natively in the browser. No WASM, no server, no plugins — just oscillators, filters, and convolution.
Sample-Accurate Timing
Lookahead scheduler using AudioContext.currentTime. Timing doesn't drift no matter how long you play.
Zero Dependencies
No React, no Tone.js, no npm. Each instrument is a single self-contained HTML file you can open offline.
Formant Synthesis
VOCA's choir uses 5-band formant filter banks with inverse-distance vowel interpolation — real IPA vowel chart positions.
Mathematical Sequencing
IRRATIONAL uses π's digit stream as an infinite non-repeating score. The melody never resolves because π is irrational.
